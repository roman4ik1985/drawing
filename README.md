# Учебное двумерное черчение

Статическое веб-приложение для учебного двумерного черчения по приложенному техническому заданию. Проект не требует сборки: достаточно открыть [index.html](C:/drawing/index.html) в браузере или запустить локальный HTTP-сервер в каталоге `C:\drawing`.

## Что реализовано

- Главное меню с разделами: `Файл`, `Правка`, `Вид`, `Рисование`, `Редактирование объектов`, `Размеры`, `Инструменты`, `Привязка`, `Настройки`, `Справка`.
- Панель инструментов быстрого доступа.
- Рабочая область на основе `SVG` с сеткой, линейками, масштабированием и панорамированием средней кнопкой мыши.
- Построение: линия, ломаная, прямоугольник, окружность, точка, текст.
- Редактирование: выделение, перемещение мышью, удаление, копирование, поворот, масштабирование, зеркальное отражение, смещение, разделение линии, базовые команды обрезки и продления для пары линий.
- Размеры: линейный, горизонтальный, вертикальный, радиус, диаметр.
- Инструменты: измерение расстояния, измерение угла между линиями, определение координат точки, встроенный калькулятор.
- Привязки: к сетке, конечным точкам, серединам, пересечениям, центрам, ортогональный режим.
- Документы: новый, открыть JSON, сохранить JSON, экспорт DXF, SVG, PNG, TIFF, печать.
- Поддержаны листы документа: нижняя панель вкладок, переключение, добавление, переименование и удаление листов.
- Команда `Сохранить как` открывает выбор форматов: `DXF / DWG / SVG / PNG / TIFF`.
- При поднятом локальном сервисе `DWG` автоматически становится доступен в меню и в `Сохранить как`.
- Автосохранение в `localStorage`.
- Справка и демонстрационный чертёж.

## Ограничения текущего MVP

- Для настоящего `DWG` нужен локальный backend и внешний конвертер.
- Обрезка и продление поддерживаются только для двух выбранных линий.
- Параллельное смещение реализовано для линий, прямоугольников и окружностей.
- Измерение угла рассчитано на линии и ломаные.
- Открытие документов реализовано через собственный JSON-формат приложения.

## Быстрый запуск

1. Откройте [index.html](C:/drawing/index.html) в браузере.
2. Либо запустите из `C:\drawing`:

```powershell
python -m http.server 8000
```

3. После запуска откройте [http://localhost:8000](http://localhost:8000).

## Локальный DWG backend

Фронтенд ожидает сервис на `http://127.0.0.1:8765`.

1. Скопируйте [dwg-service.example.json](C:/drawing/backend/dwg-service.example.json) в `C:\drawing\backend\dwg-service.json`.
2. Для `ODA File Converter` по умолчанию используется путь `C:\Program Files\ODA\ODAFileConverter\ODAFileConverter.exe`. Если программа установлена в другое место, измените первый элемент `converter.command_template`.
3. Запустите:

```powershell
powershell -ExecutionPolicy Bypass -File C:\drawing\backend\start-dwg-service.ps1 -Config C:\drawing\backend\dwg-service.json
```

Если `dwg-service.json` ещё не существует, `start-dwg-service.ps1` создаст его из шаблона и завершится. После этого отредактируйте конфиг и запустите команду повторно.

Для локальной проверки без CAD-конвертера можно использовать [dwg-service.mock.json](C:/drawing/backend/dwg-service.mock.json). Он проверяет связку frontend/backend, но не создаёт настоящий DWG.

## Горячие клавиши

- `Ctrl+S` сохранить
- `Ctrl+O` открыть
- `Ctrl+Z` отменить
- `Ctrl+Y` повторить
- `Delete` удалить выделение
- `Enter` завершить ломаную
- `Esc` отменить текущую команду
